# APK Backup
> 어플리케이션을 쉽고 간편하게 백업/복원/추출할 수 있게 해주는 어플리케이션입니다!

[![License: MIT](https://img.shields.io/badge/Lecense-MIT%202.0-blue.svg)](https://github.com/prigic/APK-Backup-Android/blob/master/LICENSE)

## Features
- 설치된 어플리케이션 백업
- 설치된 어플리케이션 APK 추출
- 백업된 APK어플리케이션 복원
